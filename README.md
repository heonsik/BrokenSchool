# 부서진 학교

Roblox 생존 탈출 게임 `부서진 학교`의 첫 구현 버전이다.

현재 구현된 것:

- 밤의 부서진 학교 맵 자동 생성
- 시작 교실
- 탈출문 목표 지점
- 목표 도착 시 `탈출 성공!` 메시지
- 눈괴물 1마리
- 눈괴물 순찰
- 가까운 플레이어 추격
- 잡히면 시작 지점으로 이동
- 슬라이딩 기능
- 기본 HUD
- 배경/괴물 근접/잡힘/탈출 효과음
- 퓨즈 3개 수집 목표
- 발전기 복구 후 탈출문 개방
- 발전기 복구 후 괴물 속도가 올라가는 마지막 추격
- 손전등 아이템과 F키 토글
- 에너지 음료 속도 버프
- 보호 부적 1회 방어
- 로비 NPC 안내
- 사물함 은신 구역
- 깨진 유리 소음 함정
- 도서관, 급식실, 발전기실 구조물
- 북쪽 별관, 남쪽 별관, 체육관, 운동장, 과학실, 음악실, 미술실, 방송실, 보건실
- PathfindingService 기반 괴물 추격
- 손전등으로 괴물을 잠시 느리게 만드는 눈부심 기능
- 잡힘 빨간 화면 효과
- 괴물 근접 위험 경고
- 최종 추격 붉은 조명 연출
- 탈출 완료 플레이어 잡힘 제외 처리
- 반투명 도서관 귀신 NPC
- 귀신이 가까이 오면 차가운 기운 메시지와 약한 소음 유발

## Roblox Studio에서 확인하는 방법

가장 쉬운 방식은 `BrokenSchool.rbxlx` 파일을 Roblox Studio로 여는 것이다.

1. `BrokenSchool.rbxlx`를 Roblox Studio로 연다.
2. Play를 눌러 테스트한다.

Studio 편집 화면에서는 `StaticPreviewMap` 프리뷰 맵이 보인다. `Play`를 누르면 서버 스크립트가 프리뷰를 제거하고 실제 게임용 맵을 자동 생성한다.

맵 부지는 약 `700 x 460` studs로 확장되어 초기 `132 x 82` 맵보다 가로와 세로가 모두 5배 이상 크다.

개발 중 소스 파일을 계속 수정하면서 확인하려면 Rojo로 Roblox Studio에 동기화한다.

1. Rojo를 설치한다.
2. 이 폴더에서 `rojo serve`를 실행한다.
3. Roblox Studio에서 새 Baseplate를 연다.
4. Studio의 Rojo 플러그인에서 서버에 연결한다.
5. Play를 눌러 테스트한다.

Rojo가 설치되어 있지 않다면, 먼저 아래 명령으로 설치할 수 있다.

```powershell
cargo install rojo
```

## 조작

- 이동: Roblox 기본 이동
- 점프: Space
- 슬라이딩: LeftControl 또는 C
- 손전등: F
- 모바일: 화면의 Slide 버튼

## 테스트 목표

1. 시작 교실에서 생성되는지 확인한다.
2. 로비 NPC 가까이 가면 안내 메시지가 나오는지 확인한다.
3. 퓨즈 3개를 먹으면 진행도가 올라가는지 확인한다.
4. 퓨즈 없이 탈출문에 가면 잠김 메시지가 나오는지 확인한다.
5. 퓨즈 3개를 모은 뒤 발전기를 밟으면 탈출문이 열리는지 확인한다.
6. 탈출문에 닿으면 `탈출 성공!` 메시지가 나오는지 확인한다.
7. 손전등을 먹고 F키로 켜고 끌 수 있는지 확인한다.
8. 에너지 음료를 먹으면 잠깐 빨라지는지 확인한다.
9. 보호 부적이 있으면 한 번 잡힘을 막는지 확인한다.
10. 사물함 안에서 괴물이 못 찾는지 확인한다.
11. 깨진 유리를 밟으면 괴물이 더 멀리서 반응하는지 확인한다.
12. LeftControl 또는 C로 슬라이딩이 되는지 확인한다.
13. 손전등을 괴물 쪽으로 비추면 괴물이 느려지는지 확인한다.
14. 잡혔을 때 화면이 빨갛게 번쩍이는지 확인한다.
15. 괴물이 가까워지면 위험 경고가 나오는지 확인한다.
16. 발전기 복구 후 조명이 붉게 바뀌고 최종 추격 메시지가 나오는지 확인한다.
17. 도서관 귀신이 떠다니고 가까이 가면 차가운 기운 메시지가 나오는지 확인한다.
18. 북쪽 별관, 남쪽 별관, 체육관, 운동장, 발전기실, 대형 탈출문까지 이동 가능한지 확인한다.
